A **React-based workout planner** that generates a **personalized weekly workout plan** for users based on their fitness goals and available workout days.  

Designed to be **responsive**, **mobile-friendly**, and visually appealing with **animated workout cards** and goal badges.

---

## Features

- Select **fitness goal**: Fat Loss / Muscle Gain  
- Select **number of workout days** (3-5)  
- Click **Generate Plan** to view a **personalized weekly workout plan**  
- Clean, modern **UI with animated cards**  
- Mobile-friendly and responsive design

---

## Tech Stack

- **React** (functional components + hooks)  
- **Vite** (fast build & dev environment)  
- **CSS** for styling and animations
